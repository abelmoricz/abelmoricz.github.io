Deep Dream Graphical User Interface:
I’m intrigued by Google’s Deep Dream but was frustrated by the long rendering time and lack of visual organizations. I decided to build 3 different levels of complexity. First, a menu with small thumbnails gives an initial impression of each feature. Second, native feature visualization creates a small (300x300) picture within seconds. Third, an overlaying of the feature on an image or native feature which can take the good part of a minute. I added image classification and have the intention to expand the project to make feature creation more intuitive.

Website for Biogärtnerei Florian Kamlah:
Badly in need of a website redesign I built the website for this family farm. Initially I wanted to use a framework like Angular but I soon realized that the ideal solution was not the most technically sophisticated one. An all in one hosting and web design service proved to be the most economical and requires the least effort to update. Even though I didn’t get to implement and expand my Angular knowledge I did provide the most value added solution to the customer, for which, of course, I was compensated.
